# Data Analytics Project

## 📌 Project Overview
The goal of this project is to simulate a corporate-grade end-to-end data analytics workflow, demonstrating the ability to translate raw data into strategic business intelligence by:

✅ Data Preparation,Modeling & Exploratory Data Analysis (Python): Clean and transform the raw dataset for analysis.

✅ Data Analysis (SQL): Simulate business transactions, and run queries to extract insights on customer segments, loyalty, and purchase drivers.

✅ Visualization & Insights (Power BI): Build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.

✅ Report and Presentation: Write a clear project report summarizing your key findings and business recommendations. Prepare a presentation that visually communicates insights and actionable recommendations to stakeholders.

## Project Workflow / Steps

### 1. Data Loading

* Imported the dataset using Pandas
* Checked data structure and column information

### 2. Exploratory Data Analysis (EDA)

* Explored data patterns and trends
* Identified missing values and duplicate records
* Analyzed important columns and metrics

### 3. Data Cleaning

* Handled null values
* Removed duplicate records
* Fixed inconsistent or incorrect data
* Prepared cleaned dataset for analysis

### 4. SQL Analysis

* Imported cleaned data into MySQL Server
* Executed SQL queries for:

  * Filtering data
  * Aggregation
  * Sorting
  * Grouping and analysis

### 5. Power BI Dashboard

* Built an interactive dashboard in Power BI
* Created charts and KPI visuals
* Added filters and slicers for better analysis

### 6. Report & Presentation

* Prepared a detailed project report
* Created a PowerPoint presentation explaining:

  * Project objectives
  * Workflow
  * Analysis process
  * Key insights and results

## How to Run the Project

### 1. Clone the Repository

```bash
git clone <repository-link>
```

### 2. Install Required Library

```bash
pip install pandas
```

### 3. Run Python File

```bash
python analysis.py
```

### 4. Run SQL Queries

* Open MySQL Server
* Import the cleaned dataset
* Execute SQL query files

### 5. Open Power BI Dashboard

* Open the `.pbix` file in Power BI Desktop
